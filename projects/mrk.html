MRK - Maison Rabih Kayrouz Website

A custom e-commerce integration for the luxury fashion brand, enabling seamless product synchronization between Shopify and the public website.

Project details

The goal was to build a robust integration layer for the Maison Rabih Kayrouz (MRK) website, powered by Magnolia CMS—a content management system used to create and manage website pages. My job was to develop a synchronization module that automatically pulls product data from Shopify every 10 minutes, ensuring only modified, added, or removed items are updated. This intelligent delta-sync approach minimized server load and kept the frontend catalog consistently up-to-date without manual intervention. Additional tasks included creating responsive templates and implementing a high-performance search functionality tailored to the brand’s product structure.

Technologies used

  • Magnolia CMS / Spring
  • Shopify REST and GraphQL API
  • Java 11
  • Freemarker (templating)
  • HTML, CSS, Javascript
